# Обработка обмена с сайтом для 1С:УНФ

## Создание обработки в конфигураторе

### 1. Создать внешнюю обработку
Файл → Новый → Внешняя обработка → Имя: `ОбменССайтом`

### 2. Реквизиты обработки
Добавить реквизиты:
- `АдресСервера` (Строка, 500) — URL API
- `КлючAPI` (Строка, 100) — ключ авторизации

### 3. Табличные части
Добавить табличную часть `СопоставлениеПолей`:
- `ПолеСайта` (Строка, 100)
- `ОписаниеПоля` (Строка, 200)
- `Обязательное` (Булево)
- `РеквизитНоменклатуры` (Строка, 200)

Добавить табличную часть `СопоставлениеЦен`:
- `ВидЦеныСайта` (Строка, 100)
- `ОписаниеЦены` (Строка, 200)
- `ТипЦен1С` (СправочникСсылка.ВидыЦен)

Добавить табличную часть `ТоварыДляВыгрузки`:
- `Номенклатура` (СправочникСсылка.Номенклатура)
- `ИдСайта` (Число, 10, 0)
- `Выгружать` (Булево)

### 4. Форма обработки
Создать управляемую форму с двумя страницами (вкладками):
- **Сопоставление** — настройка полей и цен
- **Обмен** — выгрузка/загрузка товаров

### 5. Элементы формы

**Вкладка "Сопоставление":**
- Поле ввода `АдресСервера`
- Поле ввода `КлючAPI`
- Кнопка "Загрузить структуру сайта"
- Таблица `СопоставлениеПолей` (колонки: Поле сайта, Описание, Обязательное, Реквизит 1С)
- Таблица `СопоставлениеЦен` (колонки: Вид цены сайта, Описание, Тип цен 1С)

**Вкладка "Обмен":**
- Группа отбора номенклатуры (по группе справочника)
- Кнопка "Заполнить список"
- Таблица `ТоварыДляВыгрузки`
- Кнопка "Выгрузить на сайт"
- Кнопка "Загрузить с сайта"
- Кнопка "Связать с сайтом" (отправить UUID обратно)
- Поле `Лог` (текстовый документ)

### 6. Модули
Скопировать код из файлов:
- `МодульФормы.bsl` → Модуль формы обработки
- `МодульОбъекта.bsl` → Модуль объекта обработки
