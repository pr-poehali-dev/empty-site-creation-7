## Агент

**Идея:** AI-ассистент (цифровой сотрудник) для бизнеса, работающий на отдельном VPS-сервере (~5 000–8 000 ₽/мес) с локальной LLM (Llama 3). Подключается к системе через API бэкенд-функций.

### Задачи агента:
1. **Мониторинг цен** — раз в 2 дня анализирует остатки, выбирает товары с остатком >10 шт, ищет минимальную цену на Ozon/WB/Яндекс.Маркет (браузерный агент, эмуляция человека — Playwright + LLM), корректирует розничную цену (Базовая × коэффициент уценки)
2. **Анализ остатков и распродажи** — анализ сроков поступления, предложение распродаж для залежавшегося товара по определённым правилам
3. **Контроль заявок и оплат** — отслеживание неоплаченных заявок, оповещение менеджеров
4. **Составление заявок оптовикам** — предложение состава заявки на основе предпочтений оптовика и истории заказов
5. **Оповещение продавцов** — уведомления об изменениях цен

### Архитектура:
- **Платформа (poehali.dev):** база данных + API (бэкенд-функции) + интерфейс для людей
- **VPS-сервер:** агент с локальной LLM + Playwright + LangChain, обращается к API платформы
- API платформы входит в тариф, дополнительной оплаты нет — платим только за VPS

### Порядок:
1. Сначала строим «тело» — база, API, интерфейс (на платформе)
2. Потом подключаем «мозг» — агента на VPS
3. Без агента система работает — менеджеры пользуются интерфейсом вручную
