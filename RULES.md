## Термины проекта

### Категории пользователей
- **Владелец** — определяется по номеру телефона, один (пока). Имеет полный доступ
- **Управленцы** — имеют доступ в админ-панель, интерфейс зависит от роли
- **Обычные пользователи** — отдельный модуль авторизации (будет позже)

### Роли управленцев
- Управляющий
- Менеджер опта
- Менеджер розницы
- Продавец

### Статусы управленцев
- `not_authorized` — владелец добавил номер, человек ещё не привязал Telegram
- `pending` — привязал Telegram, владелец получил уведомление в Telegram, ждёт авторизации
- `authorized` — владелец заполнил данные (Имя, Фамилия, Роль) и авторизовал

### Терминология каталога
- **Номенклатура** — общее название для одинаковых товаров (например «Холодильник Samsung RB37»)
- **Товар** — конкретная единица, может совпадать по наименованию/артикулу, но отличаться по состоянию (новый, б/у, витринный и т.д.). Таблица товаров — позже
- **Категория** — иерархическая классификация номенклатуры (дерево с вложенностью)
- **Кнопка «Добавить товар»** — пока открывает форму номенклатуры; позже — выбор: создать номенклатуру или добавить товар к существующей

### Терминология заявок
- **Заявка** — оптовый заказ, создаётся менеджером опта или управляющим
- **Позиция заявки** — строка: номенклатура + количество + цена + сумма
- **Оптовик** — покупатель, указывается текстом в заявке

### Статусы заявок — 3 независимых трека

**Трек выполнения:** `new` (Новая) → `confirmed` (Подтверждена) → `shipped` (Отгружена)
- Меняется кнопкой «Следующий шаг» в карточке заявки

**Трек оплаты:** `not_paid` (Не оплачена) → `partially_paid` (Частично оплачена) → `paid` (Оплачена)
- Рассчитывается автоматически по сумме платежей (таблица `order_payments`)
- Платёж: сумма + способ (cash / card_transfer / bank_account) + комментарий
- Если сумма платежей < суммы заявки → partially_paid, если >= → paid

**Итоговый статус:** `В работе` или `Завершена` (completed)
- Автоматически: если `shipped` + `paid` → `completed`
- При откате оплаты completed → shipped

**Удаление:** кнопка «Удалить» = мягкое удаление (status = archived), показывается в архиве

### Доступ к модулям
- **Каталог** — доступен всем ролям (владелец, все управленцы)
- **Заявки** — видят: владелец, управляющий, менеджер опта. Создают: управляющий, менеджер опта (владелец — только просмотр)
- **Цена закупа** (`price_purchase`) — видна ТОЛЬКО владельцу
